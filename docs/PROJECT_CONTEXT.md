# Project Context - Enreach Digital Showroom

## Company Vision & Mission
Enreach Concepts is a premier luxury building materials provider. The vision of the Enreach Digital Showroom is to redefine how high-end construction materials—including luxury tiles, custom doors, premium plumbing, and interior finishes—are discovered, customized, and purchased across Africa. Rather than relying on outdated printed catalogs, this platform acts as an interactive, digital-first virtual showroom.

## Product Philosophy
*   **Aesthetics Over Commodity**: Products are curated assets representing craftsmanship, texture, and visual depth.
*   **Virtual Experience**: Customers can preview premium building materials in situ within stylized luxury installation contexts.
*   **No E-Commerce Shopping Carts**: This is not an open e-commerce platform. Purchases of high-value structural assets require direct consulting, quotes, and customer care. The checkout process is replaced with a WhatsApp Lookbook Inquiry flow.

## Showroom & AI Philosophy
*   **Dynamic Visualizations**: Using Google Imagen 3, the platform automatically generates studio packshots and situated context installations for every catalog item.
*   **PIM-Driven Automation**: Uploading a single specification sheet initiates an automated pipeline that drafts luxury marketing copy, designs SEO parameters, writes FAQs, generates professional renderings, and indexes terms.

## Customer Personas & Journey
*   **Premium Architects & Designers**: Use the showroom to discover textures, copy product codes, download brochures, and build client lookbooks.
*   **Luxury Homeowners**: Browse visual layouts, save curated collections, and request direct consultations.
*   **Commercial Builders**: Need structured catalogs, validation parameters, and immediate quote pathways.

## Core Workflows
1.  **Admin Ingestion**: Admin uploads spec sheet -> system assigns unique sequential product code -> pipeline enqueues AI tasks.
2.  **AI Pipeline**: Analysis of specifications -> generation of descriptions -> generation of studio & context images -> full text search indexing.
3.  **Discovery & Inquiry**: User filters feed -> views details -> adds to a Collection -> requests quote via pre-filled WhatsApp lead message.
